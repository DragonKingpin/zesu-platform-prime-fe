# 鼠标拖拽调整事件长度功能测试指南

## 功能概述
经济日历现在支持鼠标拖拽调整事件长度功能，用户可以通过拖拽事件右侧边缘来改变事件持续时间。

## 测试步骤

### 1. 基础拖拽测试
1. 打开经济日历页面（http://localhost:8082/）
2. 查看日历中的经济事件（不同颜色的事件代表不同影响级别）
3. 找到任何有持续时间的事件（通常是多天事件）
4. 将鼠标悬停在事件右侧边缘，会出现一个圆形的调整手柄

### 2. 拖拽操作测试
1. **开始拖拽**：
   - 点击并拖拽事件的右侧圆形调整手柄
   - 界面会显示"拖拽右侧边缘调整事件时长"的提示信息
   - 右上角会出现实时预览提示框

2. **拖拽过程**：
   - 拖拽时右上角会显示：
     - 当前选择的事件时长（天）
     - 起始日期信息
     - 动态更新的持续天数
   - 被覆盖的日期范围会有高亮显示
   - 高亮区域有脉冲动画效果

3. **完成拖拽**：
   - 释放鼠标完成调整
   - 系统会显示"事件时长已调整为 X 天"的确认消息
   - 事件实际持续时间已更新

### 3. 视觉反馈测试
1. **调整手柄样式**：
   - 默认显示蓝色渐变圆形
   - 鼠标悬停时变为紫色渐变并放大
   - 有脉冲动画效果

2. **拖拽预览样式**：
   - 右上角半透明黑色提示框
   - 蓝色渐变文字显示天数
   - 底部白色三角箭头指示器
   - 高亮区域有虚线边框和渐变背景

3. **动画效果**：
   - 提示框淡入动画
   - 高亮区域脉冲动画
   - 手柄悬停缩放动画

### 4. 边界情况测试
1. **最小/最大时长**：
   - 尝试将事件调整为1天（最短）
   - 尝试调整为较长时长（如30天）

2. **交叉日期处理**：
   - 测试跨月份的事件调整
   - 测试年末到年初的调整

3. **错误处理**：
   - 在拖拽过程中切换视图
   - 在拖拽过程中刷新页面

## 预期结果

### 成功指标
- ✅ 拖拽手柄可见且响应鼠标交互
- ✅ 拖拽过程中实时显示天数预览
- ✅ 拖拽结束后事件时长实际改变
- ✅ 视觉效果流畅，有适当的动画
- ✅ 用户反馈信息清晰明确

### 注意事项
- 拖拽功能主要适用于月视图中的多天事件
- 单天事件可能没有明显的调整手柄
- 拖拽范围受限于当前月份视图

## 技术实现亮点
- 基于FullCalendar的eventDurationEditable功能
- 自定义拖拽状态管理和预览系统
- 实时DOM操作和CSS动画
- 响应式设计适配不同屏幕尺寸
- 完整的错误处理和用户反馈机制

## 发现问题
如果遇到任何问题，请检查：
1. 浏览器控制台是否有JavaScript错误
2. 拖拽手柄是否正确显示
3. 预览提示框是否正常弹出
4. 事件时长是否实际更新

测试完成后，经济日历的拖拽调整功能应该能够提供直观、流畅的用户体验。
